# Tetris

This is a classic Tetris game built with vanilla JavaScript. Playable in the browser with no dependencies.

---

## Features

- Fully functional Tetris gameplay
- Piece rotation and collision detection
- Score tracking
- Game over detection and restart

---

## How to Play

1. Clone or download the repository
2. Open `index.html` in any modern browser
3. Use the keyboard to play:

| Key | Action |
|-----|--------|
| ← → | Move piece left/right |
| ↑ | Rotate piece |
| ↓ | Soft drop |
| Space | Hard drop |

---

## Getting Started

```bash
# Clone the repository
git clone https://github.com/Oluwadeuce/tetris.git
cd tetris

# Open in browser
open index.html
```
